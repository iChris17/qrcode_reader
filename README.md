# QR Code Scanner React Native App

QR Code Scanner App for NicaSource Assesment

## Installation

Use yarn package to install dependencies

```bash
yarn start
```

## Usage

You can use either the android command or iOS.

```bash
yarn android
yarn ios
```

## ESLint test

```bash
yarn lint
```

## Unit Testing

```bash
yarn test
```
